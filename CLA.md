## How to sign

Until an automated CLA-signing bot (e.g., [CLA Assistant](https://cla-assistant.io/))
is configured for this repository, contributors agree to this CLA by
adding a `Signed-off-by:` line to every commit message:

```
Signed-off-by: Your Real Name <your.email@example.com>
```

(Use `git commit -s` to add this automatically.)

By including a `Signed-off-by` line, You certify that You have read and
agree to this CLA for that contribution. Pull requests with commits
missing a `Signed-off-by` line will be asked to amend before merging.

For corporate contributions where an employer holds IP rights, please
contact the Project Owner (`yxren_CN@outlook.com`) to arrange a Corporate
CLA before submitting.
