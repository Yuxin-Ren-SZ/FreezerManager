# Code of Conduct

## Our Pledge

We pledge to make participation in FreezerManager respectful and harassment-free
for everyone.

## Expected Behavior

- Be respectful and constructive.
- Accept feedback gracefully.
- Focus criticism on technical work, not people.
- Be mindful that this project may involve sensitive research and clinical
  contexts.

## Unacceptable Behavior

Harassment, threats, discriminatory language, sexualized attention, doxxing, and
deliberate disruption are not acceptable.

## Enforcement

Report conduct concerns to Yuxin Ren at `yxren_CN@outlook.com`. Maintainers may
remove comments, reject contributions, or restrict participation when behavior
violates this code.

This code is adapted from the Contributor Covenant, version 2.1.
