  -e POSTGRES_DB=fmgr_test -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=test \
  -p 5432:5432 postgres:16

Env gotchas seen this session (both fixed/worked-around, but watch for recurrence):
- ninja `manifest 'build.ninja' still dirty` = a tracked file has a **future
  mtime** (bulk copy/checkout skew). Fix: `find . -newermt now -print` then
  `touch` them to now.
- `Duplicate preset: "conan-debug"` = stale `CMakeUserPresets.json`; `run-tests.sh`
  now disables conan generating it and removes any stale copy.
